Jumping in the HTML

by Hae Jin Song
in html on 2016-01-26

Wouldn't it be cool if we can jump around within the same HTML document? For example, if we have a really long page (with a lot of large images), we wouldn't want to scroll all the way back and forth in order to move around the document.

There is a simple solution to this problem!

<a href="#targetName">some content</a>
