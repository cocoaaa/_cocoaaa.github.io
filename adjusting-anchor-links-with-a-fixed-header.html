Adjusting anchor links with a fixed header

by hjsong
in css on 2016-01-26 | tags: css

When there is a header that follows you as you scroll down/up the page, it may overlap with other tags. For example, it can overlap with the anchor tags as in the example below.

Using a bit of html and css, we can avoid this overlapping. You can play with this jsfiddle as well.

Originally we had:

<div class="header" style="position: fixed; top: 0; border: solid" >
This is the header.
</div>

<div class="nav">
  <ul>
    <li><a href="#section1">Section 1</a></li>
    <li><a href="#section2">Section 2</a></li>
    <li><a href="#section3">Section 3</a></li>
  </ul>
</div>

<!--<span class="anchor" id="section1"></span>-->
<div id="section1">
Jump here to start section 1
</div>

<!--<span class="anchor" id="section2"></span>-->
<div id="section2" >
Jump here to start section 2
</div>

<!--<span class="anchor" id="section3"></span>-->
<div id="section3">
Jump here to start section 3
</div>

Without any modification, if we click on 'Section 2', the content "Jump here to start section 2" overlaps with the header.

However, if we add a '' element before each target destination '

', then we can control the format of '' to avoid this overlap. Let's add them in HTML, and modify them in CSS.

HTML

<div class="header" style="position: fixed; top: 0; border: solid" >
This is the header.
</div>

<div class="nav">
  <ul>
    <li><a href="#section1">Section 1</a></li>
    <li><a href="#section2">Section 2</a></li>
    <li><a href="#section3">Section 3</a></li>
  </ul>
</div>

<span class="anchor" id="section1"></span>
<div class="target" id="section1">
Jump here to start section 1
</div>

<span class="anchor" id="section2"></span>
<div class="target" id="section2" >
Jump here to start section 2
</div>

<span class="anchor" id="section3"></span>
<div class="target" id="section3">
Jump here to start section 3
</div>

CSS

.anchor {
  border:solid red; /*for demostration */
  display: block;
  height: 30px; /*Same height as the header's */
<!--  visibility: hidden; /*Uncomment it to use */-->
}

Or

div.target {
    padding-top: 30px; /* Same height as the header's*/
}
