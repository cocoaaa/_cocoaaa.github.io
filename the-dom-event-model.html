The DOM Event Model

by Hae Jin Song
in js on 2016-02-11

Resource

1. Unlike most attribute, which take a string value, event handlers must be set using a function reference.

    :::html
    <input id="button1" type="button" value="start"
            onclick=function(){
                if(this.value ==="start"){
                    this.value = "end";
                } else{
                    this.value = "start";
                }
            }/>
    :::

-

    :::js
    var button1 = document.getElementById("button1");
    function highlight(event){
        this.style.backgroundColor='black';
        this.style.color='white';
    }

    button1.onmouseover = highlight;
    button1.onmouseout = normal;

    function normal(event){
        this.style.backgroundColor='white';
        this.style.color='black';
    }
    :::

Note
1. handler functions (e.g.highlight and normal) 2. The events are assigned to a function reference.
If we use highlight() instead of highlight, the returned object of the highlight() would be assigned as the handler. 3. We don't have to define the function before referring to it.
For example, we referred to normal before the definition.
