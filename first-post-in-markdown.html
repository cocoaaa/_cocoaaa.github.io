Syntax Highlighting

Code can be highlighted by specifiying the name of the language following ':::'.

def sayHi():
    print "Hi!"

We cannot use '```' unlike in Markdown. Here is another example in c++. More language notations can be found in this article.

void writeBlog(int hours):
    cout << "Writing like a flowing water"

Woo, that is pretty!

Block comments

We can also add blockquotes using '>'.

Shakespeare says, oh Romeo Juliet!

Blockquotes can be nested as following:

Apple is not banana.

Banana is not cinnamon.

Caramel is not donut.

Horizontal lines

Bold, italic, ~~strikethrough~~ are all well-supported.
